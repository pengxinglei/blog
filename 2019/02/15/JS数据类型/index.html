JS数据类型 原始值与引用数据类型

  • 原始值 ==> 基本数据类型
    1.number
    2.string
    3.boolean
    4.undefined
    5.null

  • 复杂数据类型/引用数据类型 object
    1.Array
    2.Object
    3.Function

typeof 检测数据类型
Array类型

  • 第一种使用Array构造函数
    var arr = new Array()

  • 第二种字面量表示法
    var arr = []

Object类型

  • 第一种Object构造函数

  • 第二种字面量

简单字面量
var obj = {};
obj.name = ‘aa’;

嵌套字面量
var obj = {name:’aa’}

JSON数据键值都是双引号 当键里面有空格、“-”等特殊符号以及关键字等都要引号

第三种工厂方式 驼峰式命名

function createObj(name){
var obj = new Object();
obj.name = name;
return obj;
};
var p = createObj(“a”);

第四种构造函数 驼峰式命名，首字母大写

function CreateObj(name){
this.name = name;
};
var p = new CreateObj(‘a’)

function类型

函数声明
function a(x,y){
return x+y;
}

函数表达式
var a = function(x,y){
return x+y;
}

区别：声明调用可以在前后调用，表达式必须在后调用

值类型与引用类型的区别：值类型不会改变原数组，引用类型会改变原数组
