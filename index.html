VUE

是一套渐进式框架 数据驱动

vue 是尤雨溪个人开发的
jquery是 操作dom

m model 模型 （当前视图可用的数据）
v view 视图1 渲染html
c controller 控制器 处理/加工model
vm 双向数据绑定

mvc
mvvm模式

vue属于mvvm模式 单应用界面

vue写法

new Vue{(
el:’#my’, //挂载元素 element vue1.0可以挂载在body上，2.0不可以
data{ //数据
name:’xz’,
list:{name：’xz’}
},
mounted:function(){//生命周期 挂载完成

},
methods:{//方法
    add:function(name){//可以不写chan
        if(name == ' ') return;
        this.lists.push({name:name}); //注意这里要加this  数据追加到最前面unshift()
        this.name='';
    },
    del:function(index){
        this.lists.splice(index,1);//删除当前  index位置  1是个数
    }
},
filters{//过滤器
    stateFilter:function(type){
        switch(type){
            case '0':
                return '未采购’;
            case '1':
                return '采购中’;
            case '2':
                return '已采购’;
            default:
                return '未采购’;
        }
    }
}

)}

“1”表达式 输出数据
