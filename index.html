MAP

map

####键值对 类似对象 传统对象只能用字符串当作键名，map的键可以是其他的类型

map上的方法

var v = Map();//创建方法
.set(key，vlu)//存储方法
.get()//获取值得方法
.has()//判断方法
.delete()
.clear()
.size()//长度

for of 获取值

for (let key of m1.keys()){
}
for (let key of m1.values()){
}
for (let [k,v] of m1()){
}

运用扩展运算符取值

var s = […keys()]
var s = […values()]
var s = […m1()]

转换类型

1、map转数组 扩展运算符
2、数组转map new Map(s) new Map([…s])
3、转为对象
var o = {}
for (let [k,v] of m1){
o[k] = v;
}
