JS类型

JS类型

数据类型 七大类型

基本数据类型：number 、 string 、 boolean 、 undefined 、 null
复杂数据类型/引用数据类型 (object array) 、 (object function)
null是空的对象 => var obj = null
所以 undefined === null //false

引用类型 三大类型

1、array类型
a)使用array构造函数 var arr = new Array()
b)字面量表示法 var arr = [a,b,c]

2、object类型
a)object构造函数 var obj = new Object()
b)字面量表示法
简单字面量 var obj = {}
嵌套字面量 var obj = {
name:’xiaoming’,
age:20
}
c)工厂方式
function createObj(name){
var obj = new Object()
obj.name = name
obj.showName = function(){function(){console.log(this.name)}}
return obj
}
var p = createObj(‘a’)
p.showName()

d)构造函数 首字母大写 驼峰式

function CreateObj(name){
        this.name = name
        this.showName = function(){function(){console.log(this.name)}}
    }
    var p = new CreateObj('a')
    p.showName()

2、function类型

a)函数声明//在前后调用都可以
sum()
function sum(){}
sum()

a)函数表达式//在后面调用
var sum = function(){}
sum()

值类型和引用类型的区别
值类型，直接赋给别人，不会互相影响
引用类型会改变原来的
